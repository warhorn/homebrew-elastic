# Warhorn Elastic

## How do I install these formulae?

`brew install warhorn/elastic/<formula>`

Or `brew tap warhorn/elastic` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "warhorn/elastic"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
